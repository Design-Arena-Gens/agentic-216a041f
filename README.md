## SmartTeammates · Skyline

Вэб-реализация мода **SmartTeammates** от Skyline. Приложение демонстрирует полноценную панель Mod Options и живой боевой симулятор, который отражает влияние настроек на поведение союзных ботов.

### Основные возможности

- Настраиваемый множитель урона (от 1.0× до 2.5×) с моментальным сохранением.
- Три профиля скорости передвижения: «Стандарт», «Быстро», «Очень быстро».
- Боевой симулятор, визуализирующий, как тиммейты преследуют и отключают врагов.
- Локальное сохранение предпочтений пользователя.

### Локальный запуск

```bash
npm install
npm run dev
```

Перейдите по адресу [http://localhost:3000](http://localhost:3000), чтобы увидеть мод в действии.

### Сборка

```bash
npm run build
```

### Развёртывание

Приложение готово к деплою на Vercel:

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-216a041f
```

После публикации убедитесь, что страница доступна по адресу https://agentic-216a041f.vercel.app.
